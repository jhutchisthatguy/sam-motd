Welcome to Funky Sandbox Server!

Server Rules

  • Listen to admins and mods
  • No server crashing or exploits
  • Respect fellow players
  • Have fun and enjoy the game!

Server Information

This is a sandbox casual server where you are free to do as you wish, within reason.
Build, explore, and interact with others, but remember to respect the rules and the enjoyment of fellow players!

Server Commands

  • !motd – Shows the message of the day
  • !mount – Downloads and Mounts missing server content for you.
  • !thirdperson – Enables/Disables third person camera view.
  • !buddies – Opens the gProtect buddies menu so you can give your friends permission to interact with your props
  • !build – Enables build mode.
  • !tp – Teleports you to a player and has a cooldown of 25s. You need to type a player's name after !tp.
  • !autounstuck – Enables auto unstuck. Type the command again to disable.
  • !cleanupself – Removes all of your own props.
  • !gestures – Lists gestures/acts.
  • !cmd – Lists more commands in the chat.

Garry's Mod 64-bit Recommended

For the best performance and to use all server features (like the Media Player), we strongly recommend playing Garry's Mod in 64-bit mode. If you’re still on 32-bit, some features won’t work.
